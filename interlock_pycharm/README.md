# interlock_pycharm
## Pycharm(IDE) 와  github 연동해서 편하게 쓰자

1. 깃에서 새로운 repository를 생성하고 얻은 https url 를 Pycharm(IDE)에 입력해준다.
2. 이후 해당 repository의 폴더가 만들어지게 되는 [파일 생성/수정] > [commit] > [push] 단계를 거치면 깃에 자동으로 반영이된다.
